# github_public_repos_search

## run
1- clone this repository

2- install ruby version 2.7.4

3- install bundler `gem install bundler`

4- run `bundle install`

5- run `rails s`

------------

## About this project

This project uses Rails version 5, the main goal of this project is to search public repositories on Github.

It use Github's API to fetch public repositories, you can find the documentation for Github API here: https://docs.github.com/en/rest/reference/search

This project is very simple it uses Github API to fetch data and displays the results on the webpage


## Tests

This project use Rspec for tests.

To run the tests `rspec`
